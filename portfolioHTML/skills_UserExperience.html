User Experience

‣ 致力於讓使用者擁有更好的體驗

‣ 以設計的方法解決使用者在體驗上可能發生的障礙

‣ 以下幾個案例 會以比較 原始設計 與 再設計過後 的差異來說明

個人網站⎯瀏覽列的再設計

‣ 原設計： 專案和技能混在一起造成凌亂 → 再設計： 把專案和技能分開，讓使用者易分辨
‣ 原設計： 純粹以專案的名字不易讓使用者了解專案性質 → 再設計： 加入圖片說明
‣ 原設計： 回首頁的設計不明顯 → 再設計： 換成icon，樣式與瀏覽列區隔，使之較為明顯

瀏覽列的再設計與製作

‣ 原設計： 有些點選是直接跳畫面，有些則是顯示下拉選單。
→ 再設計： 加入符號「」讓使用者易辨認哪些具有下拉選單

‣ 原設計： 下拉選單顯示不清楚。
→ 再設計： 微調配色，讓顏色對比度提高，使之更為清晰。

選單的再設計與製作

‣ 原設計： 太多選單，容易讓使用者分不清。
→ 再設計： 透過 分析該樣物件的存在 為固定或不固定的方式，讓固定存在的物件同一個樣式，使不固定存在的物件 與固定存在的樣式 有所區隔，讓畫面更加清楚。
