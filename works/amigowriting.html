Overview

My friend runs instagram business profile which is full of quotes handwriting. There is only one link can put in the Instagram bio, so my friend asked me to do a page with links. I added a message box in it so that fans can recommend books or quotes thay liked.

Role: UI Design, Front-end Development (JavaScript, SPA), Back-end Development (Node.js, Firebase, Deploy to Heroku)

Time: 2018/10

URL: amigo writing (mobile recommended)

Front-end Development

It's a Sinple Page Application (SPA). Users can leave their message and see their message on the page without refreshing. Moreover, I track user data with lihi and Mixpanel.

Back-end Development

I use Node.js to storage data in Firebase. Moreover, I deploy it to Heroku service.
