Instagram Bio Link Tool

Add multiple links and message board to your Instagram bio

·   ·   ·

Overview

My friend runs the Instagram business profile, amigo writing. She often shares pictures with motivating quotes or books by her handwriting on Facebook, Medium and Instagram. However, Instagram only allows one external link on the bio, so my friend asked me to design and develop a page with links.

I think we need to be different from existing services such as Linktree or Shorby etc, so I proposed that adding a message board will be good if her fans can leave some messages in public and recommend books or quotes they liked.

Role:Front-end Development, Back-end Development, UI Design

Time: 2018/10

URL: amigo writing (mobile recommended)

·   ·   ·

Front-end Development

When users send their messages on our message board, they can immediately see their messages on the web page. Since the message board a Single Page Application (SPA), which means only part of the page which is required to refresh will update. Therefore, it shortens the users’ loading time. They will not aware of the refreshing. I use Vanilla JavaScript to do this.

In a traditional web application, when users click on a button, the whole page will refresh on request. Users may need to wait for the page loading. However, in a single page application (SPA), when users click on a button, only part of the page which is influenced will be refresh.

Back-end Development

I use Node.js as the backend to handle the user requests and fetch, retrieve data from servers. I use Firebase as a database for data storage. Moreover, I deploy it to Heroku service which is for free. I have applied what I have learned in NTU Information System Training Program and it’s my first time accomplishing a full-stack work.
